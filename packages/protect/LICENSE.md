NoData Protect was started open so any developer could read every line and decide for themselves whether to trust it. That principle hasn't changed, and it never will:

- **Read the code.** Every line of every release is on GitHub. Audit it. Test it.
- **Use it for yourself, free, forever.** Personal projects. Side projects. Internal company use — yes, including small businesses encrypting their own production secrets. This is not a trial. There is no "30-day clock." There is no quota that suddenly appears. If you encrypt your own data, you don't pay. Ever.

After that date, every Protect version published before then is fully open source under the Apache 2.0 license, including for the use cases listed below as "Competing Use."
